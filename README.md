# SE_project

The E-Commerce Marketplace Management System is a web-based platform designed to connect customers and sellers in a secure and efficient environment. It enables product management, inventory tracking, order processing, and secure transactions while providing role-based access for customers, sellers, and administrators. The system follows a 3-tier architecture and applies Software Engineering principles such as requirements analysis, system modeling, modular design, Agile development, testing, and documentation to ensure reliability, scalability, and maintainability.
